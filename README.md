# calc

**calc** is a command line calculator interpreter.

- supports only decimal and hexadecimal numbers
- math operators supported: `+`, `-`, `*`, `/`, `%`, `|`
- bit-wise operators: `&`, `|`, `~`
- parentheses
- comment starts with `#`

**set up**

requires cc/gcc.

```shell
$ cd calc; make
$ ./calc
```
